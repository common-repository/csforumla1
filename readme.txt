Automatically display league tables from formula1.com

== Description ==

csFormula1 will add a result table from formula1.com to your page or post. You can choose between season, team or driver. Results go back to 1950

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/csformula1` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for csformula1 via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin
